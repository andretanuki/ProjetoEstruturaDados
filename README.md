# Jogo Investigativo - Projeto de Estrutura de Dados

Este repositório contém o projeto de um Jogo Investigativo em texto, executado no terminal, desenvolvido como trabalho prático para a disciplina de Estrutura de Dados. O projeto é construído em **Java**.

## Estrutura do Projeto

O jogo demonstra a aplicação de estruturas de dados clássicas:
- **Lista Encadeada (Linked List):** Gerencia o histórico cronológico de pistas coletadas pelo jogador.
- **Árvore (Tree):** Mapeia a hierarquia lógica e dependência de pistas (quais pistas liberam acesso a outras).
